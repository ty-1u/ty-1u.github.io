GO语言教程Day4

1. GO语言中的常量

相对于变量，常量的值是恒定不变的值

1.1 使用const定义常量

常量定义方法和变量定义方法类似。例如：

const pi=3.1415926

注意，定义好常量之后必须赋值，例如：

const pi float32

pi = 3.1415

fmt.Println(pi)

这时候运行会报错。

var pi float32

pi = 3.1415

fmt.Println(pi)

而这么定义，是可以正常运行的。

1.2 常量特性

常量的值是不可改变的，例如：

const a = "1"

a = "2"

fmt.Println("a")

此时运行会报错

1.3 一次声明多个常量

const(

​ a=”1”

​ b=”2”

)

const同时声明了多个常量时，如果忽略了后面的值，则后面声明的变量和上面一行的值相同

func main() {

const (

​ A = "A"

​ B

​ C = "C"

​ D

)

fmt.Println(A, B, C, D)

}

此时代码运行结果为A A C C

1.4 const常量结合iota的使用

iota 是 golang 语言的常量计数器,只能在常量的表达式中使用。iota 在 const 关键字出现时将被重置为 0(const 内部的第一行之前)，const 中每新增一行常量声明将使 iota 计数一次(iota 可理解为 const 语句块中的行索引)。

iota主要用于一次定义多个变量，如果令常量为iota，那么此常量为0.

const (

​ a = iota

​ b

​ _/（跳过某些值）

​ _

​ c

)

fmt.Println(a, b, c)

运行结果为0 1 4

iota在声明中插队

const (

​ a = iota

​ b = 100

​ c = iota

​ d

)

fmt.Println(a, b, c, d)

运行此代码的结果为0 100 2 3。相当于b占用了1这个位置，称为在声明中插队

多个iota可以定义在一行

const (

​ a, b = iota + 1, iota + 2 //1 2

​ c, d //2 3

​ e, f//3 4

)

fmt.Println(a, b, c, d, e, f)

运行此代码，根据上面的知识，a,b的值分别为1，2，那么c,d的值就是对应位置的值再加1，所以此代码的运行结果为1 2 2 3 3 4

2. GO语言常量变量命名规则

和前面的变量命名规则类似，但是注意，GO语言区分大小写，所以age和Age是两个不同的变量或者常量。

建议命名遵循驼峰命名的规则，例如：

一个单词时，建议小写

var username=18

多个单词时组成时，单词和单词之间首字母大写

``var maxAge=18 //小驼峰表示私有`

var MaxAge=19 //大驼峰表示公有

3. GO语言的代码风格

代码每一行结束不用写分号（加入也不算错）

运算符左右建议加空格

GO语言推荐使用驼峰命名法

强制的代码风格

go fmt main.go 在终端运行，可以格式化文档，让代码风格保持一致（安装了go拓展包后不需要此代码）
