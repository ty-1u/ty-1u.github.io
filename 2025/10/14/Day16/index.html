1. Array（数组）的介绍

数组是指一系列同一类型数据的集合。

数组的类型由 **基本数据类型 和 长度 **定义

2. 数组的定义

var 数组名 [数组长度]数据类型

var arr1 [3]int

var arr2 [4]int

var str1 [3]string

fmt.Printf("arr1:%T,arr2:%T,str1:%T", arr1, arr2, str1)

打印arr1,arr2,str1三个数组的类型，结果为：

arr1:[3]int,arr2:[4]int,str1:[3]string

注意，数组的长度也是类型的一部分

数组的初始化

var arr1 [3]int

fmt.Println(arr1)

定义了一个[3]int的数组，直接打印，得到的结果为[0 0 0]

和基本数据类型保持一直，如果是int类型，结果是0，如果是string类型，结果是空

1. 方法1：一一赋值

var arr1 [3]int

arr1[0] = 1

arr1[1] = 10

arr1[2] = 24

fmt.Println(arr1)

结果为[1 10 24]

在golang中，数组长度指定之后，长度便不可再变

2. 方法2：定义和赋值一起

var arr1 = [3]int{1, 10, 24}

fmt.Println(arr1)

结果为[1 10 24]

3. 方法3：类型推导方式

arr1 := [3]int{1, 10, 24}

fmt.Println(arr1)

结果为[1 10 24]

4. 方法4：让编译器自行推断数组长度

var arr1 = [...]int{1, 10, 24}

fmt.Println(arr1)

利用[…]让编译器自行推断数组的长度

打印的结果为[1 10 24]

不能用于增加arr1的长度：

var arr1 = [...]string{"php", "nodejs", "golang", "js"}

arr1[4] = "cpp"//增加第五个数据

fmt.Println(arr1)

此时代码直接报错

打印数组长度len()

var arr1 = [...]int{1, 10, 24}

fmt.Println(len(arr1))

输出结果为3

5. 方法5：注明索引值的初始化

arr1 := [...]int{0: 1, 1: 10, 2: 20, 5: 50}//在5处赋值50

fmt.Println(len(arr1))

在0处赋值1，在1处赋值10，在2处赋值20，在5处赋值50

该代码的结果为6，表名这个数组中存储了6个int类型数字

3. 数组的循环遍历 for/for range

for循环

var arr1 = [...]string{"php", "nodejs", "golang", "js"}

for i := 0; i < len(arr1); i++ {

​ fmt.Println(arr1[i])

}

for range循环

var arr1 = [...]string{"php", "nodejs", "golang", "js"}

for _, v := range arr1 {//第一个位置是下标，第二个位置是元素的值

​ fmt.Println(v)

}

结果为

php
nodejs
golang
js

2.求出一个数组的最大值，并得到对应的下标

我的方法：

for循环

将arr中的各个数据和max比较，更大的将被放入max

var arr = [...]int{27, -43, 81, 149, 3, -7, 120, 55}

var max = 0

for i := 0; i < len(arr); i++ {

​ if arr[i] > max {

​ max = arr[i]

​ }

}

fmt.Println(max)

结果为149

for range循环

var max = 0

var index = 0

for k, v := range arr {

​ if v > max {

​ max = v

​ index = k

​ }

}

fmt.Println(max, index)

结果为149 3

3.从数组[1 ,3 ,5 , 7, 8]中找到和为8的两个元素，打印他们的下标，用()表示

首先定义两个变量储存下标，第一次循环取出所有值，第二次循环取出+1号的值，将它们分别相加，得到符合要求的下标值

我的代码：

var num = [...]int{1, 3, 5, 7, 8}

index1 := 0

index2 := 0

for i := 0; i < len(num); i++ {

​ for j := i + 1; j < len(num); j++ {

​ if num[i]+num[j] == 8 {

​ index1, index2 = i, j

​ fmt.Printf("(%v,%v)", index1, index2)

​ }

​ }

}

结果为(0,3)(1,2)
