1. for range（键值循环）

依次打印字符串的内容，用到for range

语法为 for k,v :=range var{}

其中，k是变量var对应字符的位置(以ascii码计，字母占两个位置，汉字占三个位置)，v对应的是相应位置的字符对应的编码的值，如汉字就是utf-8，字母是ascii码

依次取出str中的内容：

var str = "你好,golang"

for k, v := range str {

​ fmt.Printf("k=%v,v=%c\n", k, v)

由于输出的是编码值，用%c打印出原来的值，如果不想要k，可以采用隐匿变量的方法不打印k，输出的结果为：

k=0,v=你
k=3,v=好
k=6,v=,
k=7,v=g
k=8,v=o
k=9,v=l
k=10,v=a
k=11,v=n
k=12,v=g

循环遍历切片

var arr = []string{"php", "java", "node", "golang"}

for _, v := range arr {

​ fmt.Printf("%v ", v)

}

首先定义一个字符串类型的切片，最后的输出都是字符串

php java node golang

2. switch case

switch case语句可方便对大量的值进行条件判断

练习：判断文件类型，如果后缀名是.html，输入text/html,如果后缀名是.css 输出text/css ……

语法：

switch expression{

case condition：

}

当然，我们首先可以想到if来判断：

var ext = ".html"

if ext == ".html" {

​ fmt.Println("text/html")

} else if ext == ".css" {

​ fmt.Println("text/css")

} else if ext == ".js" {

​ fmt.Println("text/javascript")

} else {

​ fmt.Println("找不到此后缀")

}

用switch case实现上面功能：

var ext = ".html"

switch ext {

case ".html":

​ fmt.Println("text/html")

​ break

case ".css":

​ fmt.Println("text/css")

​ break

case ".js":

​ fmt.Println("text/javascript")

​ break

default:

​ fmt.Println("找不到此后缀")

}

注意，case condition的后面一定接上冒号“:” swtich变量后面紧接着花括号”{}”，在golang中，break可以写也可以不写，其他每行都要在任务执行完加default

switch case的另一种写法

将定义的变量作为局部变量放在swtich代码的里面，需要加分号分割：

switch ext := ".html";ext {

case ".html":

​ fmt.Println("text/html")

​ break

case ".css":

​ fmt.Println("text/css")

​ break

case ".js":

​ fmt.Println("text/javascript")

​ break

default:

​ fmt.Println("找不到此后缀")

}

依然可以正常运行

判断一个数是不是偶数

switch case同时给判断赋多个值：

var num = 5

switch num {

case 1, 3, 5, 7, 9:

​ fmt.Println("这是奇数")

case 2, 4, 6, 8:

​ fmt.Println("这是偶数")

这里我们给每种case都赋值多个值，每个值之间用逗号连接，依然可以正常运行

判断年龄

在switch上可以写表达式：

var age = 30

switch { //不接名称

case age < 20:

​ fmt.Println("年轻人")

case 20 < age && age < 60:

​ fmt.Println("中年人")

default:

​ fmt.Println("老年人")

}

注意，如果接的是表达式，此时switch后面不接变量名称。需要用判断时，建议用if else语句

switch的穿透 fallthrought

fallthrought语法可以执行满足条件的case的下一个case：

var age = 30

switch {

case age < 20:

​ fmt.Println("年轻人")

case 20 < age && age < 60:

​ fmt.Println("中年人")

​ fallthrough

case age > 60:

​ fmt.Println("老年人")

default:

​ fmt.Println("输入错误")

}

该代码本来只会执行 中年人，但是有fallthrought，将下一行的结果 老年人也执行了。注意，一个fallthrought只能穿透一层，也就是只会多执行紧挨着的下面一层的结果。
