1. break语句

表示跳出当前循环，可以用在for循环中 ，也可以用在switch case中

eg:

for i := 0; i < 10; i++ {

​ // if i == 2 {

​ // // break

​ // }

​ fmt.Println(i)

}

在加入了注销的代码之后，程序只打印了 0 1，break表示跳出当前的循环

for i := 0; i < 10; i++ {

​ if i == 2 {

​ break

​ }

​ fmt.Println(i)

}

fmt.Println("继续执行")

在循环后面加入代码后，break跳出循环，依旧会继续执行循环外的代码

eg:

for i := 0; i < 2; i++ {

​ for j := 0; j < 10; j++ {

​ if j == 3 {

​ break

​ }

​ fmt.Printf("i=%v,j=%v\n", i, j)

​ }

}

执行过程：

当i=0时

i=0,j=0

i=0,j=1

i=0,j=2

当i=3时，程序跳出次循环，但是无法跳出主循环

此时执行下一个主循环

当i=1时

i=1,j=0

i=1,j=1

i=1,j=2

程序结束运行

所以结果是

i=0,j=0
i=0,j=1
i=0,j=2
i=1,j=0
i=1,j=1
i=1,j=2

程序不打印j>2的值

在switch case中使用break

在case语句结束可以加上break，在golang中，即使不加上break也不会发生穿透

一次跳出多重循环

利用label标出想跳出的循环

label1:

for i := 0; i < 2; i++ {

​ for j := 0; j < 10; j++ {

​ if j == 3 {

​ break label1

​ }

​ fmt.Printf("i=%v,j=%v\n", i, j)

​ }

}

label1的命名随意，但是需要是字母开头，此时程序会跳过多层循环，最后的结果是：

i=0,j=0
i=0,j=1
i=0,j=2

2. Continue语句

结束当前循环，开始下一次的循环迭代，只能在for循环里使用

for i := 0; i < 10; i++ {

​ if i == 3 {

​ continue

​ }

​ fmt.Println(i)

}

该代码跳过了i=3的情况，继续执行下一个循环，因此打印的结果是：

0
1
2
4
5
6
7
8
9

Continue和label的配合使用

continue语句后添加标签时，表示跳过标签所标记的循环的“当前迭代”：

``label2: for i := 0; i < 3; i++ { for j := 0; j < 10; j++ { if j == 3 { continue label2 } fmt.Printf(“i=%v,j=%v\n”, i, j) } }

比如这个代码，当程序执行到i=0，j=3时，此时执行continue会跳过i=0的大循环，开始i=1的大循环，当再次执行到i=1，j=3时，程序会跳过i=1的大循环开始下一个大循环，所以最后的结果为：

i=0,j=0
i=0,j=1
i=0,j=2
i=1,j=0
i=1,j=1
i=1,j=2
i=2,j=0
i=2,j=1
i=2,j=2

3. goto(跳转到指定标签)

goto语句通过标签进行代码间的无条件跳转

var n = 30

if n > 24 {

​ fmt.Println("成年人")

​ goto label3

}

fmt.Println("aaa")

fmt.Println("bbb")

label3:

fmt.Println("ccc")

fmt.Println("ddd")

在这个代码中，定义了label3，label3会执行打印ccc，ddd的操作，当判断语句执行到goto时，会跳转到label3执行，此时运行的结果为：

成年人
ccc
ddd
