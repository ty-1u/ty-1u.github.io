1. 切片(slice)的定义

切片是一个拥有相同类型元素的可变长度的序列。它是基于数据类型做的一层封装。非常灵活，支持自动扩容

切片是一个引用类型，它的内容包含地址，长度和容量

切片的声明

切片的定义和数组类似，但是中括号[]中不放任何值

var arr []int

fmt.Printf("值是%v,类型是%T,长度是%v", arr, arr, len(arr))

结果为：值是[],类型是[]int,长度是0

可以看到，声明一个int类型的切片，它的值为nil，长度是0

声明并赋值

var arr = []int{1, 3, 54, 6, 7, 8}

切片可以包含任意数量的元素

切片的遍历

for range遍历

var arr = []int{1, 3, 54, 6, 7, 8}

for _, v := range arr {

​ fmt.Println(v)

}

for 遍历

for i := 0; i < len(arr); i++ {

​ fmt.Println(arr[i])

}

结果为：

1
3
54
6

基于数组定义切片

切片的底层是一个数组，所以我们基于数组定义切片

var arr = [6]int{1, 3, 54, 6, 7, 8}

b := arr[:] //获取数组内的所有值

fmt.Printf("值为%v,类型为%T\n", b, b)

我们可以使用 ``b := arr[:]`的方法由数组定义一个切片，此时的b就是一个切片了，代码的结果为：

值为[1 3 54 6 7 8],类型为[]int

从一个数组截取一部分作为切片

c := arr[1:4]

fmt.Printf("值为%v,类型为%T", c, c)

利用上面方法可以从数组中截取一部分作为切片，如： c := arr[1:4]表示截取1到3位置的切片，这种表达方式是左边包括而右边不包括

同理，还可以获取第一个到某一个和某一个到最后一个的切片

例如： c := arr[:4] c := arr[1:]的定义方法

切片的再切片

和基于数组的切片类似

var slice1 = []int{1, 2, 3, 4, 5, 6, 7, 8}

slice2 := slice1[1:]

fmt.Println(slice1, slice2)

结果为：[1 2 3 4 5 6 7 8] [2 3 4 5 6 7 8]

可以看到，切片的切片也可以由上面的方法得到

切片的长度和容量

切片拥有自己的长度和容量，我们可以通过使用内置的len()函数求长度，使用内置的cap()函数求切片的容量

注意：切片的长度是它包含的元素的个数

切片的容量是从第一个元素数，到其底层数组元素末尾的个数

var slice1 = []int{11, 21, 3, 4, 5, 6, 7, 8}

fmt.Println("长度",len(slice1),"容量", cap(slice1))

结果为 8 8

slice3 := slice1[1:3]//即切片容量从第一个数21计数，到8一共是7个数

fmt.Println(len(slice3), cap(slice3))

结果为2 7

slice4 := slice1[:3]

fmt.Println(cap(slice4))

结果为8，可见容量是从给的第一个数开始计数，到底层数组的末尾，之间的元素的个数

切片的本质

切片的本质就是对底层数组的封装，它包含了三个信息：底层数组的指针，切片的长度和切片的容量

例如：

相似的，这里指针指到了数字3，继续向后面数到最后一个元素7，总共是5个，所以容量是5

1. 数组是值类型

在golang中，基本数据类型和数组都是值类型

什么是值类型？

定义：
当一个变量赋值给另一个变量时，会复制值本身，两个变量互不影响

也就是说，每个变量都有自己独立的数据副本

改变变量副本的值，不会改变本身的值

如：

a := 10
b := a  // 复制了 a 的值

b = 20  // 改变 b，不会影响 a

fmt.Println("a =", a)
fmt.Println("b =", b)

对数组，也有这个结论

Go中常见值类型

类型分类 示例
基本数据类型 int, float64, bool, string
数组 [5]int{1, 2, 3, 4, 5}
结构体 struct 自定义类型

引用数据类型

改变变量副本的值，会改变变量本身的值

2. switch case

switch case语句可方便对大量的值进行条件判断

练习：判断文件类型，如果后缀名是.html，输入text/html,如果后缀名是.css 输出text/css ……

语法：

switch expression{

case condition：

}

判断年龄

在switch上可以写表达式：

var age = 30

switch { //不接名称

case age < 20:

​ fmt.Println("年轻人")

case 20 < age && age < 60:

​ fmt.Println("中年人")

default:

​ fmt.Println("老年人")

}

注意，如果接的是表达式，此时switch后面不接变量名称。需要用判断时，建议用if else语句

switch的穿透 fallthrought

fallthrought语法可以执行满足条件的case的下一个case：

var age = 30

switch {

case age < 20:

​ fmt.Println("年轻人")

case 20 < age && age < 60:

​ fmt.Println("中年人")

​ fallthrough

case age > 60:

​ fmt.Println("老年人")

default:

​ fmt.Println("输入错误")

}

该代码本来只会执行 中年人，但是有fallthrought，将下一行的结果 老年人也执行了。注意，一个fallthrought只能穿透一层，也就是只会多执行紧挨着的下面一层的结果。

2. 位运算符(了解)

位运算符对整数在内存中的二进制位进行操作

运算符 名称 示例 含义说明 适用类型
& 按位与 a & b 对应二进制位都为 1 时结果为 1 整数类型
| 按位或 a | b 两数有一个为1时结果为1 整数类型
^ 按位异或 a ^ b 对应二进制位不同为 1，相同为 0 整数类型
&^ 按位清零（AND NOT） a &^ b 将 a 中对应 b 为 1 的位清零，其余位保持不变 整数类型
<< 左移 a << 2 将 a 的二进制位左移 2 位，高位丢弃，低位补 0 整数类型
>> 右移 a >> 2 将 a 的二进制位右移 2 位，符号位保持（有符号数） 整数类型

示例：

a := 6  // 0110
b := 3  // 0011

fmt.Println("a & b =", a & b)   // 2  -> 0010
fmt.Println("a | b =", a | b)   // 7  -> 0111
fmt.Println("a ^ b =", a ^ b)   // 5  -> 0101
fmt.Println("a &^ b =", a &^ b) // 4  -> 010

也就是将十进制的数转化为二进制进行同位的运算

左移

a := 6 // 0110

a = a << 2

fmt.Println(a)

左移相当于乘以2的n次方，上面结果为24

右移

​ a := 10 // 二进制 1010` `fmt.Println(a >> 1) // 10 / 2 = 5` `fmt.Println(a >> 2) // 10 / 4 = 2 （丢掉小数 0.5）` `fmt.Println(a >> 3) // 10 / 8 = 1 （丢掉小数 0.25）

右移代表整除2的n次方，即右移只保留商的整数部分

当公式中出现常量的时候，系统会自动统一变量和常量的类型，如：

负数的取余计算：

输出结果为 -1 1 计算方法为 余数=被除数- (被除数/除数)*除数

被除数 (a) 除数 (b) 表达式 结果 说明
+5 +2 5 % 2 +1 都是正数
+5 -2 5 % -2 +1 结果跟被除数一样是正数
-5 +2 -5 % 2 -1 结果跟被除数一样是负数
-5 -2 -5 % -2 -1 结果跟被除数一样是负数

可以总结一个口诀： 余数看左边，符号随被除

也就是把被除数和除数按照绝对值运算，最后， 被除数的符号和余数的符号一致，确定余数的符号

执行

var a = 9

test…
执行

首先 a>10，产生了false，由于逻辑运算符是or，必须运行test才能知道运不运行if语句，运行test后，返回true，此时or运算通过，再运行if

运算符 名称 左边结果 是否执行右边 整体结果 口诀

|| 逻辑或 false ✅ 执行右边 看右边结果 右边真则运行，假则不运行
