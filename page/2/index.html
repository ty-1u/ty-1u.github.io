1.为什么在go语言中第一行一定要写package？

1. 程序是以 包（package） 为基本组织单位的

  • Go 的所有源文件必须属于某个包，编译器才能识别。

  • 每个 .go 文件都需要通过 package 包名 来声明自己属于哪个包

    比如

    package main声明了此文件属于main包

    注意：package main表面这个文件是一个可执行程序，而如果package后面声明了其他的变量，则是一个库，给别的文件import，类似于python的库

    这里用package main示范

package main 后面无脑接上func main(){}

类似于：

package main

func main() {

2.fmt包

fmt 包是 Go 标准库 里非常常用的一个包，名字来自 format（格式化，fmt包主要用来 **格式化输入输出，就像 C 语言里的printf/scanf，或者 Python 里的print()、input()

常用函数

输出函数

fmt.Print() → 普通输出，不换行。

fmt.Println() → 输出并换行。

fmt.Printf() → 格式化输出，可以控制显示格式。

2.1 一次输出多句话

例如fmt.Print("A", "B", "C")，输出ABC不会有空格，而fmt.Println("A", "B", "C")输出的A B C则会有空格

2.2 变量定义的两种方法

例如，我们可以用var a="aaa"来定义一个字符串，或者也可以使用类型推导方法来定义，类似于a := "aaa"注意，在go语言中，定义了的变量必须使用，不使用的话甚至会报错

Println 和 Printf的区别

fmt.Printf("%v", a)运行这行代码一样会打印出aaa

示例： 想要打印三个数字，定义如下

var a int = 10

var b int = 5

var c int = 3

用下面两种方式打印

fmt.Println("a=", a, "b=", b, "c=", c)

fmt.Printf("a=%v b=%v c=%v", a, b, c)

得到结果如下：

a= 10 b= 5 c= 3
a=10 b=5 c=3

可以看出，%v相当于占位符，后面逗号的变量给占位符赋值，printf的适用性更好

此外，Printf函数默认不换行，如果想要换行，可使用如下方法

fmt.Printf("a=%v b=%v c=%v\n", a, b, c)

2.4 关于import函数

示范代码：

在运行这个代码时突然报错# command-line-arguments .\main.go:6:1: syntax error: unexpected }, expected name or ( 有点莫名奇妙，我代码的第六行并没有多余的花括号，怀疑是复制粘贴的代码里有多余的符号，重新写了一遍代码成功运行，如果安装了go的插件，即使不写import函数，保存后也会自动加入

2.5 打印出变量类型

打印出结果为d的类型是string

Go语言学习记录Day1

Go语言教程

一. Go 语言简介

即Golang, 是Google公司于2009年11月公布的编程语言

优点：Go语言不仅拥有静态编译语言的安全和高性能，而且又达到了动态语言开发速度和易维护性。有人形容 Go语言:Go=C+Python，说明Go语言既有C语言程序的运行速度，又能达到 Python 语言的快速开发。

二. Go语言成功的项目

nsq, docker, packer等等……

三. Go语言环境搭建

1.下载安装Golang

网址：https://go.dev/dl/（需要梯子）

2.安装过程

没有特别需要注意的，点下一步即可

3.验证是否安装成功

在终端运行go version命令

看到版本好即为安装成功

3.1查看go环境

运行go env初入门不需要配置环境等，查看终端是否显示环境信息即可

说明:Go1.11版本之后无需手动配置环境变量，使用go mod管理项目，也不需要非得把项目放到GOPATH指定目录下，你可以在你磁盘的任何位置新建一个项目。
Go1.13 以后可以彻底不要 GOPATH了

四. Go语言开发工具Vscode配置

1.下载安装Vscode

https://code.visualstudio.com/ 无需梯子

安装无特殊说明，点击下一步即可

2.汉化vscode

由于默认的vscode是英文的，在拓展项中搜索language，选择中文简体，点击标蓝字体install安装

注意：安装成功之后，右下角会有一个弹窗询问你是否要更改语言，选择是，vscode会自动重启，这时候界面就是中文的了，如果没有显示，可以先卸载再次安装，等弹窗出现

3. 安装Go语言插件

Go语言插件可以作为提示帮助我们写代码

按照安装语言包的流程，可以拿来安装Go语言插件，我们在拓展框里搜索Go，排名第一的就是该插件

4.项目新建

在任意位置新建文件夹，注意命名不要出现中文和空格

比如我选择在d盘新建一个go_demo的文件用于开发

打开vscode，选择左上角的文件，选择打开文件夹

选择刚才新创建的文件夹，选择信任此文件夹。

在该文件夹下选择新建文件，命名随意，注意后缀带上.go

可以在main.go中写下自己的第一行代码，比如

package main

import (

"fmt"

)

func main() {

fmt.Println("你好 golang")

}

选择在终端运行，在终端中输入go run main.go

该代码打印了括号中的一句话 你好 golang。笔者也是首次接触golang，遇到了两个麻烦，分享解决方法：

1.Println的P没有大写

此时终端会报错expected ‘package’, found ‘EOF’

2.没有保存main.go文件就运行

此时终端同样报错expected ‘package’, found ‘EOF’

查找资料后发现，go语言对大小写区分很严格，而且文件必须要保存之后才能运行，笔者没有保存就运行，电脑会自动运行原程序，而原来程序是我们新建的文件，自然无法运行

此外，vscode的右下角会提示需要安装某些安装包，直接选择安装即可

由于某种不可预测的原因，在输出里可能会出现这样的报错代码

“stderr”: “go: github.com/haya14busa/goplay/cmd/goplay@v1.0.0: github.com/haya14busa/goplay/cmd/goplay@v1.0.0: Get "https://proxy.golang.org/github.com/haya14busa/goplay/cmd/goplay/@v/v1.0.0.info\“: dial tcp 142.250.217.113:443: connectex: A connection attempt failed because the connected party did not properly respond after a period of time, or established connection failed because connected host has failed to respond.\n”

表明此时安装包未安装成功

解决方法：

使用国内 Go 模块代理

打开cmd,运行：

go env -w GO111MODULE=on
go env -w GOPROXY=https://goproxy.cn,direct

再重新安装gopls运行：

go install golang.org/x/tools/gopls@latest

可以检测是否安装成功：

gopls version

若显示版本号，就说明安装成功了，此时重启vscode，再任意go文件里输入.fmt，看是否有提示比如Println这种的，有则说明成功了，此时保存文件时，go会自动删除没有声明的变量，大家也可以通过这个来判断
