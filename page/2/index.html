GO语言教程Day5

1. Golang数据类型介绍

Golang语言分为基本数据类型和复合数据类型

基本数据类型有：

整形，浮点型，布尔型，字符串

复合数据类型有：

数组，切片，结构体，函数，map，通道，接口等

2. 整形

var num int = 10

fmt.Printf("类型是%T", num)

定义了int类型，赋值不能是其他形式

整形分为两大类，一类是有符号整型，一类是无符号整型：

有符号整型按长度分为：int8，int16，int32，int64

对应的无符号整型为：uint8，uint16，uint32，uint64

类型 范围（数值） 数学表达 占用空间 是否有符号
int8 -128 到 127 -2^7 到 2^7-1 1 个字节 有
int16 -32,768 到 32,767 -2^15 到 2^15-1 2 个字节 有
int32 -2,147,483,648 到 2,147,483,647 -2^31 到 2^31-1 4 个字节 有
int64 -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807 -2^63 到 2^63-1 8 个字节 有
uint8 0 到 255 0 到 2^8-1 1 个字节 无
uint16 0 到 65,535 0 到 2^16-1 2 个字节 无
uint32 0 到 4,294,967,295 0 到 2^32-1 4 个字节 无
uint64 0 到 18,446,744,073,709,551,615 0 到 2^64-1 8 个字节 无

找一个合适的长度，避免溢出

关于字节：

字节也叫byte，是计算机数据的基本储存单位，8bit=1Byte 1024Byte=1KB, 在电脑里一个中文占两个字节。

unsafe.Sizeof包

查看不同长度类型的包

var num1 int8 = 98

var num2 int64 = 98

fmt.Println(unsafe.Sizeof(num1))

fmt.Println(unsafe.Sizeof(num2))

unsafe.Sizeof()结果是以字节计数，所以输出的结果为：1 8

int类型转换

var num1 int8 = 1

var num2 int16 = 1

fmt.Println(num1 + num2)

同样长度的变量运算才能正常运行，因此上面的代码是运行不了的，可以对变量进行强制的转换，比如：

var num1 int8 = 1

var num2 int16 = 1

fmt.Println(int16(num1) + num2)

此时代码正常运行，可以得到输出： 2

转换类型时候，尤其是高位数据向低位数据转换时，不要超过该类型的范围

var num1 int16 = 130

fmt.Println(int8(num1))

此时代码虽然能正常运行，但是得到的结果却是-126，很明显是130的补码

数字字面量语法

num := 30

fmt.Printf("num等于%v,类型为%T\n", num, num)

fmt.Println(unsafe.Sizeof(num))

计算机默认以电脑的位数储存信息，比如我用的是64位电脑，因此使用int时，默认使用64位储存信息

Go1.13 版本之后引入了数字字面量语法，这样便于开发者以二进制、八进制或十六进制浮点数的格式定义数字，例如：

num := 30

// fmt.Println(unsafe.Sizeof(num))

fmt.Printf("num等于%b\n", num)

fmt.Printf("num等于%o", num)

可以输出：

num等于11110
num等于36

GO语言教程Day4

1. GO语言中的常量

相对于变量，常量的值是恒定不变的值

1.1 使用const定义常量

常量定义方法和变量定义方法类似。例如：

const pi=3.1415926

注意，定义好常量之后必须赋值，例如：

const pi float32

pi = 3.1415

fmt.Println(pi)

这时候运行会报错。

var pi float32

pi = 3.1415

fmt.Println(pi)

而这么定义，是可以正常运行的。

1.2 常量特性

常量的值是不可改变的，例如：

const a = "1"

a = "2"

fmt.Println("a")

此时运行会报错

1.3 一次声明多个常量

const(

​ a=”1”

​ b=”2”

)

const同时声明了多个常量时，如果忽略了后面的值，则后面声明的变量和上面一行的值相同

func main() {

const (

​ A = "A"

​ B

​ C = "C"

​ D

)

fmt.Println(A, B, C, D)

}

此时代码运行结果为A A C C

1.4 const常量结合iota的使用

iota 是 golang 语言的常量计数器,只能在常量的表达式中使用。iota 在 const 关键字出现时将被重置为 0(const 内部的第一行之前)，const 中每新增一行常量声明将使 iota 计数一次(iota 可理解为 const 语句块中的行索引)。

iota主要用于一次定义多个变量，如果令常量为iota，那么此常量为0.

const (

​ a = iota

​ b

​ _/（跳过某些值）

​ _

​ c

)

fmt.Println(a, b, c)

运行结果为0 1 4

iota在声明中插队

const (

​ a = iota

​ b = 100

​ c = iota

​ d

)

fmt.Println(a, b, c, d)

运行此代码的结果为0 100 2 3。相当于b占用了1这个位置，称为在声明中插队

多个iota可以定义在一行

const (

​ a, b = iota + 1, iota + 2 //1 2

​ c, d //2 3

​ e, f//3 4

)

fmt.Println(a, b, c, d, e, f)

运行此代码，根据上面的知识，a,b的值分别为1，2，那么c,d的值就是对应位置的值再加1，所以此代码的运行结果为1 2 2 3 3 4

2. GO语言常量变量命名规则

和前面的变量命名规则类似，但是注意，GO语言区分大小写，所以age和Age是两个不同的变量或者常量。

建议命名遵循驼峰命名的规则，例如：

一个单词时，建议小写

var username=18

多个单词时组成时，单词和单词之间首字母大写

``var maxAge=18 //小驼峰表示私有`

var MaxAge=19 //大驼峰表示公有

3. GO语言的代码风格

代码每一行结束不用写分号（加入也不算错）

运算符左右建议加空格

GO语言推荐使用驼峰命名法

强制的代码风格

go fmt main.go 在终端运行，可以格式化文档，让代码风格保持一致（安装了go拓展包后不需要此代码）

GO语言学习记录Day3

1. 变量类型

变量(Variable)的功能是存储数据。不同的变量保存的数据类型可能会不一样。经过半个多世纪的发展，编程语言已经基本形成了一套固定的类型，常见变量的数据类型有:整型、浮点型、布尔型等。
Go 语言中的每一个变量都有自己的类型，并且变量必须经过声明才能开始使用

2. GO 语言中变量的声明

Go 语言变量名由字母、数字、下划线组成，其中首个字符不能为数字。Go 语言中关键字和保留字都不能用作变量名。
Go 语言中的变量需要声明后才能使用，同一作用域内不支持重复声明。 并且 Go 语言的变量声明后必须使用。

需要说明的是，只是不支持重复声明，但仍可以重新赋值，如：

a := 1

a = 2

fmt.Println(a)

此时打印出的结果正常显示2

如果打印代码

a := 1

a := 2

fmt.Println(a)

此时便会报错

2.1 var变量声明

2.1.1 单变量赋值

格式： var 变量名称 类型 如 var uername string ，类型可以省略。

在未进行赋值时，打印值为空

var username string = "张三"

fmt.Println(username)

赋值之后，可正常打印出张三

也可以这样定义

var usename string

username = "张三"

注意，此时变量类型不能省略，否则会报错

2.1.2 多变量赋值

多变量同种类型

例如：

var a1,a2 string

a1="aaa"

a2="aaaaa"

fmt.Println(a1,a2)

输出可以正常打印出 aaa aaaaa,注意，因为声明了a1 a2均为字符串，不可以对a1 a2赋值其他类型的数据

多变量不同类型

例如：

var (

​ name string

​ age int

​ sex string

)

name = "张三"

age = 20

sex = "男"

fmt.Println(name, age, sex)

可以正常打印出张三 20 男

2.2 短变量声明法

格式：变量名：=表达式 例如 username := "张三"

短变量只能用于局部变量声明，不能用于全局变量声明，即在func main里面的短变量可以正常引用，但不能引用func main外的全局变量，需要声明全局变量可以用上面的var方法

短变量声明法声明多个变量，例如

a, b, c := 1, 2, 3

a, b, c := 1, 2, "三"

上面两种声明方法都是正确的，说明短变量声明法声明多个变量可以声明不同类型的，十分方便

2.fmt包

fmt 包是 Go 标准库 里非常常用的一个包，名字来自 format（格式化，fmt包主要用来 **格式化输入输出，就像 C 语言里的printf/scanf，或者 Python 里的print()、input()

常用函数

输出函数

fmt.Print() → 普通输出，不换行。

fmt.Println() → 输出并换行。

fmt.Printf() → 格式化输出，可以控制显示格式。

2.1 一次输出多句话

例如fmt.Print("A", "B", "C")，输出ABC不会有空格，而fmt.Println("A", "B", "C")输出的A B C则会有空格

2.2 变量定义的两种方法

例如，我们可以用var a="aaa"来定义一个字符串，或者也可以使用类型推导方法来定义，类似于a := "aaa"注意，在go语言中，定义了的变量必须使用，不使用的话甚至会报错

Println 和 Printf的区别

fmt.Printf("%v", a)运行这行代码一样会打印出aaa

示例： 想要打印三个数字，定义如下

var a int = 10

var b int = 5

var c int = 3

用下面两种方式打印

fmt.Println("a=", a, "b=", b, "c=", c)

fmt.Printf("a=%v b=%v c=%v", a, b, c)

得到结果如下：

a= 10 b= 5 c= 3
a=10 b=5 c=3

可以看出，%v相当于占位符，后面逗号的变量给占位符赋值，printf的适用性更好

此外，Printf函数默认不换行，如果想要换行，可使用如下方法

fmt.Printf("a=%v b=%v c=%v\n", a, b, c)

Go语言教程

一. Go 语言简介

即Golang, 是Google公司于2009年11月公布的编程语言

优点：Go语言不仅拥有静态编译语言的安全和高性能，而且又达到了动态语言开发速度和易维护性。有人形容 Go语言:Go=C+Python，说明Go语言既有C语言程序的运行速度，又能达到 Python 语言的快速开发。

二. Go语言成功的项目

nsq, docker, packer等等……

三. Go语言环境搭建

1.下载安装Golang

网址：https://go.dev/dl/（需要梯子）

2.安装过程

没有特别需要注意的，点下一步即可

3.验证是否安装成功

在终端运行go version命令

看到版本好即为安装成功

3.1查看go环境

运行go env初入门不需要配置环境等，查看终端是否显示环境信息即可

说明:Go1.11版本之后无需手动配置环境变量，使用go mod管理项目，也不需要非得把项目放到GOPATH指定目录下，你可以在你磁盘的任何位置新建一个项目。
Go1.13 以后可以彻底不要 GOPATH了

四. Go语言开发工具Vscode配置

1.下载安装Vscode

https://code.visualstudio.com/ 无需梯子

安装无特殊说明，点击下一步即可

2.汉化vscode

由于默认的vscode是英文的，在拓展项中搜索language，选择中文简体，点击标蓝字体install安装

注意：安装成功之后，右下角会有一个弹窗询问你是否要更改语言，选择是，vscode会自动重启，这时候界面就是中文的了，如果没有显示，可以先卸载再次安装，等弹窗出现

3. 安装Go语言插件

Go语言插件可以作为提示帮助我们写代码

按照安装语言包的流程，可以拿来安装Go语言插件，我们在拓展框里搜索Go，排名第一的就是该插件

4.项目新建

在任意位置新建文件夹，注意命名不要出现中文和空格

比如我选择在d盘新建一个go_demo的文件用于开发

打开vscode，选择左上角的文件，选择打开文件夹

选择刚才新创建的文件夹，选择信任此文件夹。

在该文件夹下选择新建文件，命名随意，注意后缀带上.go

可以在main.go中写下自己的第一行代码，比如

package main

import (

"fmt"

)

func main() {

fmt.Println("你好 golang")

}

选择在终端运行，在终端中输入go run main.go

该代码打印了括号中的一句话 你好 golang。笔者也是首次接触golang，遇到了两个麻烦，分享解决方法：

1.Println的P没有大写

此时终端会报错expected ‘package’, found ‘EOF’

2.没有保存main.go文件就运行

此时终端同样报错expected ‘package’, found ‘EOF’

查找资料后发现，go语言对大小写区分很严格，而且文件必须要保存之后才能运行，笔者没有保存就运行，电脑会自动运行原程序，而原来程序是我们新建的文件，自然无法运行

此外，vscode的右下角会提示需要安装某些安装包，直接选择安装即可

由于某种不可预测的原因，在输出里可能会出现这样的报错代码

“stderr”: “go: github.com/haya14busa/goplay/cmd/goplay@v1.0.0: github.com/haya14busa/goplay/cmd/goplay@v1.0.0: Get "https://proxy.golang.org/github.com/haya14busa/goplay/cmd/goplay/@v/v1.0.0.info\“: dial tcp 142.250.217.113:443: connectex: A connection attempt failed because the connected party did not properly respond after a period of time, or established connection failed because connected host has failed to respond.\n”

表明此时安装包未安装成功

解决方法：

使用国内 Go 模块代理

打开cmd,运行：

go env -w GO111MODULE=on
go env -w GOPROXY=https://goproxy.cn,direct

再重新安装gopls运行：

go install golang.org/x/tools/gopls@latest

可以检测是否安装成功：

gopls version

若显示版本号，就说明安装成功了，此时重启vscode，再任意go文件里输入.fmt，看是否有提示比如Println这种的，有则说明成功了，此时保存文件时，go会自动删除没有声明的变量，大家也可以通过这个来判断
